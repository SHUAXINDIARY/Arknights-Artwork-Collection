# User Data Catch - Chrome 插件

用于抓取 Twitter/X 和微博用户数据的 Chrome 插件。

## 功能

- **Twitter 列表抓取**: 自动滚动抓取 Twitter 关注列表所有用户数据
- **Twitter 单用户抓取**: 抓取当前 Twitter 用户主页的数据
- **微博列表抓取**: 自动滚动抓取微博关注列表所有用户数据
- **微博单用户抓取**: 抓取当前微博用户主页的数据

## 安装步骤

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `UserDataCatch` 文件夹

## 图标说明

插件需要 PNG 格式的图标文件：
- `icon16.png` (16x16)
- `icon48.png` (48x48)  
- `icon128.png` (128x128)

你可以：
1. 使用任意图片编辑工具创建这三个尺寸的图标
2. 或者在 manifest.json 中删除 icon 相关配置（插件会使用默认图标）

## 使用方法

1. 打开 Twitter 关注列表页面 或 微博关注页面
2. 点击浏览器右上角的插件图标
3. 选择对应的抓取按钮
4. 等待抓取完成，数据会自动复制到剪贴板

## 数据格式

抓取的数据为 JSON 格式：

```json
[
  {
    "nickname": "用户昵称",
    "avatar": "头像URL",
    "profile": "主页URL",
    "bio": "个人简介"
  }
]
```

## 注意事项

- 列表抓取会自动滚动页面，请勿中途操作
- 抓取完成后数据自动复制到剪贴板
- 建议在网络良好的环境下使用
